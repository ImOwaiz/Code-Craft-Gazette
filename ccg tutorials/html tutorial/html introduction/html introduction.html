HTML Introduction

Some important points to remember about HTML :
  • HTML stands for Hyper Text Markup Language.
  • HTML is the basic building block of any websites and it is used to structure the websites.
  • HTML is not a programming language, it is a markup language used to format text, images, and links on a webpage.
  • HTML is platform-independent and can be used on any device or browser.
  • HTML is easy to learn, and it is the first step to dive in Web-Development

A Simple HTML Code

Example Code

<!DOCTYPE html>
<html>
<head>
<title>Your Page Title</title>
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

Explaination about this code:

  • The Document Type Declaration <!DOCTYPE html>, a way to tell the browser this is an HTML page, and which version of HTML we are using.
  • The <html> element is the main and important element of an HTML page
  • The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
  • The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • The <h1> element defines a large heading
  • The <p> element defines a paragraph
